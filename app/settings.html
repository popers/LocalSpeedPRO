Jak skonfigurować Client ID/Secret?
  1. Wejdź na Google Cloud Console i utwórz nowy projekt.
  2. Włącz API "Google Drive API" w bibliotece.
  3. W "Credentials" utwórz "OAuth Client ID" (Typ: Web application).
  4. Skopiuj Redirect URI z pola powyżej i wklej w konsoli Google.
